### Stock Explorer (Boutiqueinfantil.com.br)


Just publishing here on github a good android app used on a former company that I owned. Maybe the source code can help on my futures projects.
This is an [**Android**](https://developer.android.com) project with some good use cases of [ButterKnife](https://github.com/JakeWharton/butterknife), [Retrofit](https://github.com/square/retrofit), [Picasso](https://github.com/square/picasso) and [Barcodescanner](https://github.com/dm77/barcodescanner)
